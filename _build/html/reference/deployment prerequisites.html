Deployment Prerequisites¶

The following re-requisites have been identified for a standard install

Hardware & Operating System¶

Component Value
Hypervisor
  • VMware Esxi 6 (preferred)
  • VirtualBox
OS
  • Ubuntu Server 14.04 (preferred)
  • SUSE Linux Enterprise Server 11 Service Pack 3
  • CentOS 7

Software¶

Component Value
Software
  • Jdk 1.8+
  • ImageMagick 6.9+
  • ffmpeg-3.2+
  • ExifTool-10.36
  • ghostscript-9.20+

Component Value

  • 443 (https) Security Controls
