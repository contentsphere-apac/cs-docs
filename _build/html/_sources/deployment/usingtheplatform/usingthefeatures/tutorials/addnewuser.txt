==============
Add New Users
==============
Manage different roles or profiles within the system by adding new users.


How-to: Add New Users
======================
To do this, follow these steps:


|

* Click **Administration**.

.. image:: /feature/_images/mdmimages/cs_addingnewusers_clicksettings.PNG
|
|

* Click **Partner Administration**.

.. image:: /feature/_images/mdmimages/cs_addingnewusers_clickpartneradmnistration.PNG
|
|

* Click **Users** on the left sidebar.

.. image:: /feature/_images/mdmimages/cs_addingnewusers_clickusers.PNG
|
|

* Then, click "+" to add a user.

.. image:: /feature/_images/mdmimages/cs_addingnewusers_clickplus.PNG
|
|

* Enter data in the fields.

.. image:: /feature/_images/mdmimages/cs_addingnewusers_enterdetails.PNG
|
|

* Then, click **Create**.

.. image:: /feature/_images/mdmimages/cs_addingnewusers_clickcreate.PNG
|
|

* View the added user.


.. image:: /feature/_images/mdmimages/cs_addingnewusers_viewuseradded.PNG
|
|


Icon for the user can also be added. To do this, follow the steps below:

      * Click **Add Icon** and select an image icon from your computer.

      .. image:: /feature/_images/mdmimages/cs_addingnewusers_clickaddicon.PNG
      |
      |


      * View the added icon.

      .. image:: /feature/_images/mdmimages/cs_addingnewusers_viewaddedicon.PNG
      |
      |

|
