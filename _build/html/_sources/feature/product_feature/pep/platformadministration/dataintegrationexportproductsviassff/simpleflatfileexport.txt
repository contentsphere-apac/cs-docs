========================
Simple Flat File Export
========================
Simple Flat File Export is a feature that manages *File-based Endpoints* so that data can be exported in a flat file format (e.g. *.xls/.csv*).


===================     =======================================================================================================================================================================
**Description:**        * An admin can organize the file endpoints in the system and can use them to export data in *.xls or .csv* format.
                        * The system, by using this feature, allows organizations to have a system with multiple outbound endpoints following their own processes.

**Related topics:**     * :doc:`Manage Endpoints for a System </feature/product_feature/pep/platformadministration/systemprofilesetupandmanagesystemspecificdata/manageendpointsforasystem>`
                        * :doc:`Manage Outbound Endpoint </feature/product_feature/pep/platformadministration/dataintegrationmanageoutboundendpoint/manageoutboundendpoint>`
                        * :doc:`Manage Outbound Process </feature/product_feature/pep/platformadministration/dataintegrationmanageoutboundprocess/manageoutboundprocess>`

**Application:**        * An admin creates endpoints, which will help a user to off-board data into sheets. These sheets can be viewed outside the platform and be reused.
                        * *Example:*

                            An administrator can set up endpoints that can be used by any user or supplier to export data in sheets which can then be used in other systems.

===================     =======================================================================================================================================================================


How to do: Simple Flat File Export
===================================
To do a simple flat file export, follow these steps:

|

#. Log in and go to *Administration*.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_login.PNG
|
|

#. Click on the "+" button to create a class.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_clickplusbutton.PNG
|
|

#. Enter the name and select the class type, then click on the **Create** button.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_entername.PNG
|
|

#. Click on the "+" icon to add property collection.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_addpropertycollection.PNG
|
|

#. Select the property collection and click on the **Apply** button.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_addpropertycollection2.PNG
|
|

#. Go to *Explore* and click on **PIM**.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_clickexplore.PNG
|
.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_clickpim.PNG
|
|

#. Create products with the newly created class and click on the **Save** button.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_createproducts.PNG
|
|

#. Go back to *Administration*.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_gobacktoadmin.PNG
|
|

#. Click on the **Data Integration** tab.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_clickdataintegration.PNG
|
|

#. Click on **Workflow** then click on the "+" icon to create a process.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_createprocess.PNG
|
|

#. Enter the name for the process and set the type as "Business Process Workflow", then click on the **Create** button.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_clickcreate.PNG
|
|

#. Drag and drop the components from the *Graph* section.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_draganddrop.PNG
|
|

#. Click the **Connect** icon to connect the components.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_connecticon.PNG
|
.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_connecticon2.PNG
|
.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_connecticon3.PNG

#. Click on the **Change type** icon.

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_changetype.PNG

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_changetype2.PNG

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_changetype3.PNG

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_clickmappings.PNG

.. image:: /feature/_images/pepimages/cs_simpleflatfileexport_enternamemappings.PNG
|

|
|


#. Click on the **Download** icon. View the information sheet once the download is complete.

|
|

#. Click on the endpoint to view the list of exported products in the *Catalog* tab.

|
|

#. View the downloadeded sheet.

|
