==================
Create New Product
==================

How to do: Create New Products
==============================
To do this, follow these steps:


|

#. 1.	Click  ALL from top right.

.. image:: /feature/_images/mdmimages/createnewprod_clickall.PNG
|
|

#. Hover over the "+" icon...and select the desired product(in this case we select a single product).

.. image:: /feature/_images/mdmimages/createnewprod_clicksingleprod.PNG
|
|

#. Click on **Loose Bundle**.

.. image:: /feature/_images/mdmimages/managedisplays_clickloose.PNG
|
|

#. Enter the  necessary information.

.. image:: /feature/_images/mdmimages/managedisplays_enterinformation.PNG
|
|

#.  Click  **Add products**.

.. image:: /feature/_images/mdmimages/managedisplays_clickaddproducts.PNG
|
|

#.   Click the green button to add the required products (button turns to blue when selected), and click  **Add All**.

.. image:: /feature/_images/mdmimages/managedisplays_selectandaddall.PNG
|
|

#.    To add more products, click **Return**, and repeat the previous steps.

.. image:: /feature/_images/mdmimages/managedisplays_addedsuccesfullyandreturn.PNG
|
.. image:: /feature/_images/mdmimages/managedisplays_imageadded.PNG
|
|

#.   To add an image, click  **Relationships** tab.

.. image:: /feature/_images/mdmimages/managedisplays_gotorel.PNG
|
|

#.   Hover over on the "+" icon and click  **Add Entity**.

.. image:: /feature/_images/mdmimages/managedisplays_addentity.PNG
|
|


#.   Hover over on the "+" icon and click  **Image**.

.. image:: /feature/_images/mdmimages/managedisplays_hoverimage.PNG
|
.. image:: /feature/_images/mdmimages/managedisplays_imagesuccess1.PNG
|
|

#.   Click **Add Image** again.

.. image:: /feature/_images/mdmimages/managedisplays_imageagain.PNG
|
.. image:: /feature/_images/mdmimages/managedisplays_imagesuccess2.PNG
|
|

#.   View bundle on the top section of the page.

.. image:: /feature/_images/mdmimages/managedisplays_imageready.PNG
|
|
