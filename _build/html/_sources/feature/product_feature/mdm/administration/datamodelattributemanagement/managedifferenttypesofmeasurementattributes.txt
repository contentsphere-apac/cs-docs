Measurement Type: Volume
=========================
This feature example allows an administrator to set an attribute as *Volume*.

===================     ===============================================================================================================================================================

**Key Term:**           * **Attribute:** a data field which describes a product, can also use ‘property’.

**Description:**        * This feature allows the setting of an attribute type as *Volume*.
                        * This feature makes an attribute type as *Volume* and applies the appropriate default unit for volume (e.g. *Gallon, Cubic meter, Liter*).

**Related topics:**     * :doc:`Manage Different Types of Attributes </feature/product_feature/mdm/administration/datamodelattributemanagement/managedifferenttypesofattributes>`
                        * :doc:`Manage Concatenated Attributes </feature/product_feature/mdm/administration/datamodelattributemanagement/manageconcatenatedattributes>`
                        * :doc:`Manage Calculated Attributes </feature/product_feature/mdm/administration/datamodelattributemanagement/managecalculatedattributes>`

**Application:**        * Volume measurement type can set the volume of a liquid product (e.g. *Beverages*) and also set its corresponding unit.
                        * *Example:*

                            An administrator can set an attribute of a Red Bull product to volume so that a user can easily enter the volume of the product and select
                            a unit (from a drop-down list).

===================     ===============================================================================================================================================================


How-to: Measurement Type: Volume
-------------------------------------
To set an attribute as a volume, follow these steps:

|

* Go to *Administration*.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_gotoadmin.PNG
|
|


* Go to the *Data Model* tab.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_gotodatamodel.PNG
|
|


* Click on *Measurement* under the *Properties* section.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_clickonmeasurement.PNG
|
|


* Click the '+' icon to create an attribute.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_clickoncreatebutton.PNG
|
|


* Enter the name of the volume attribute and click on the **Create** button.

.. image:: /feature/_images/mdmimages/cs1_managemeasurementvolume_enterthename.PNG
|
|


* Check that the attribute has been added successfully.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_checkthattheattribute.PNG
|
|


* Under the measurement type, search and select *Volume* in the drop-down menu.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_searchandselectvol.PNG
|
.. image:: /feature/_images/mdmimages/cs_manageattributevolume_searchandselectvol2.PNG
|
|


* Select the appropriate unit for the volume attribute in the *Default Unit* column (e.g. *Gallon, Cubic meter, Liter*).

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_selecttheunit.PNG
|
|


* Complete the other attribute fields: *Default Value, Precision, Placeholder, Description, Tooltip, Searchable, Filterable, Sortable, Availability, Grid Editable, and Read Only*.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_completetheattributes.PNG
|
|


* Click on the **Save** button to save the settings.

.. image:: /feature/_images/mdmimages/cs_manageattributevolume_clickonsave.PNG
|
|


.. image:: /feature/_images/mdmimages/line-png-32.PNG
