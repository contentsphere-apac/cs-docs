====================================
Manage Different Types of Attributes
====================================
The Manage Different Types of Attributes is a feature that allows an administrator to manage and set up attributes in the system.


====================     ===============================================================================================================================================================
**Key Term:**             * **Property:** a property is a field used to describe a product. It comes in several different types.

**Description:**          * This feature can create and manage different types of attributes (properties) for various data. This will restrict
                            the attribute to the selected type only.

**Related topics:**       * :doc:`Manage Different Types of Measurement Attributes </feature/product_feature/mdm/administration/datamodelattributemanagement/managedifferenttypesofmeasurementattributes>`

**Application:**          * The feature will allow an administrator to set up the attributes in the system that will help other users
                            put relevant information to the product data fields.
                          * *Example:*

                              An administrator sets up a *Date* type attribute for a data field. If a general user adds an information to that field, a calendar will pop up
                              and the user can select a date value using the calendar.

====================     ===============================================================================================================================================================


How to do: Manage Different Types of Attributes
=================================================
To do this admin feature, follow the steps below:

|

#. Log in and go to *Administration*.

.. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_login.PNG
|
|


#. Go to the *Data Model* tab and then to *Properties*.

.. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_gotodatamodel.PNG
|
|


#. Select a specific type of attribute from the selection.

.. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_selectaspecifictype.PNG
|
|


#. For a particular attribute type, refer to its respective section below:

|

**Type: Text**


  #. Click on **Text** and then click the "+" icon to create an attribute.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_text1.PNG
  |
  |

  #. Enter the attribute name and then click on the **Create** button.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_text2.PNG
  |
  |

  #. View the text attribute in runtime.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_text3.PNG
  |
  |

|

**Type: Number**


  #. Click on **Number** and then click the "+" icon to create an attribute.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_number1.PNG
  |
  |

  #. Enter the attribute name and then click on the **Create** button.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_number2.PNG
  |
  |

  #. View the number attribute in runtime.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_number3.PNG
  |
  |

|

**Type: Price**

  #. Click on **Price** and then click the "+" icon to create an attribute.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_price1.PNG
  |
  |

  #. Enter the attribute name and then click on the **Create** button.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_price2.PNG
  |
  |

  #. Set the *Default Unit*.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_price3.PNG
  |
  |

  #. Provide the *Default Value, Precision, Placeholder, Description, and Tooltip* and then set the options for the attribute.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_price4.PNG
  |
  |

  #. View the price attribute in runtime.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_price5.PNG
  |
  |

|

**Type: Date**

  #. Click on **Date** and then click the "+" icon to create an attribute.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_date1.PNG
  |
  |

  #. Enter the attribute name and then click on the **Create** button.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_date2.PNG
  |
  |

  #. Set also the values for *Default Value, Placeholder, and Description*, if available.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_date3.PNG
  |
  |

  #. View the date attribute in runtime.

  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_date4.PNG
  |
  .. image:: /feature/_images/mdmimages/cs_managedifftypesofattributes_date5.PNG
  |
