=================
Manage Affinities
=================
Manage Affinities lets you show the right products to the right persona.


====================    =============================================================================================================================================================
**Key Term:**           * **Tags:** properties that describe the products.

**Description:**        * Criteria of relevancy must be established when presenting products since different personas can have different preferences or affinities.
                        * Even products carry affinities like some products are suitable during winter while some are intended for summer.
                        * Affinity-based tags allow you to show the products that correspond to a persona.

**Related topics:**     * :doc:`Manage LOVs </feature/product_feature/mdm/administration/datamodeltagmanagement/managelovs>`
                        * :doc:`Property Collections </feature/product_feature/mdm/administration/datamodelpropertycollections/propertycollections>`

**Application:**        * Show the relevant products by applying affinity-based tags.
====================    =============================================================================================================================================================


How-to: Manage Affinities
============================
To manage affinities, follow these steps:

|

#. Log in and click on the **Administration** button.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_login.PNG
|
|


#. Click on **Properties** and select **LOV**.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_clickproperties.PNG
|
.. image:: /feature/_images/mdmimages/cs_manageaffinities_selectlov.PNG
|
|


#. Click on the "+" icon to create a new tag (LOV).

.. image:: /feature/_images/mdmimages/cs_manageaffinities_clickplusicon.PNG
|
|


#. Create a tag type "Yes Neutral". (Example: Gender > Male / Female)

.. image:: /feature/_images/mdmimages/cs_manageaffinities_gender.PNG
|
|


#. Create a tag type "Yes Neutral No". (Example: Hobbies > Football / Baseball / Basketball)

.. image:: /feature/_images/mdmimages/cs_manageaffinities_hobbies.PNG
|
|


#. Create a tag type "Range". (Example: Weather > Summer / Winter / Spring)

.. image:: /feature/_images/mdmimages/cs_manageaffinities_weather.PNG
|
|


#. Create a tag type "Ruler". (Example: Income > Low / Medium / High)

.. image:: /feature/_images/mdmimages/cs_manageaffinities_income.PNG
|
|


#. Go to *Property Groups* and create a property collection.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_createpropertycollection.PNG
|
.. image:: /feature/_images/mdmimages/cs_manageaffinities_createpropertycollection2.PNG
|
|


#. Drag and drop the tags to the property collection.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_draganddrop1.PNG
|
|


#. Go to *Classes* and select a class, then click on the "+" icon.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_gotoclasses.PNG
|
|


#. Select the property collection and click on the **Apply** button.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_selectpropertycollection.PNG
|
|


#. Click on the **Explore** button and go to *PIM*.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_gotoclasses.PNG
|
.. image:: /feature/_images/mdmimages/cs_manageaffinities_gotopim.PNG
|
|


#. Hover over the "+" icon and click on **Single Article**.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_hover.PNG
|
|


#. Go to the *Property Collection* tab and view the property collection.

.. image:: /feature/_images/mdmimages/cs_manageaffinities_propertycollectiontab.PNG
|
.. image:: /feature/_images/mdmimages/cs_manageaffinities_view.PNG
|
|
