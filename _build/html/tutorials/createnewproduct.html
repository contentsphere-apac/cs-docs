Create a New Product¶

The system allows effective management of products in the system. Products can be created, deleted, and edited.

How-to: Create New Products¶

To do this, follow these steps:


  • Click Explore from the top right and go to PIM.




  • Hover over the “+” icon and select the desired product (in this demo, we select Smartphone Accessories).




  • Enter the necessary information.






  • Go back to Overview tab and click Add Taxonomy to add taxonomy to the product.




  • Taxonomy menu appears. Select the desired taxonomy (in this demo, we select Accesories).


  • View the selected taxonomy.


  • Click Relationships tab to add an image to the product.


  • Click on “+” to add an entity.


  • Click the select checkbox to add the desired image (button turns to blue when selected), then click Add.




  • Drag the selected image to the right pane can also be performed to add the image.


  • To add more image, click Back, and repeat the previous steps.
