Property Collections¶

Property Collection feature allows an administrator to group the attributes and tags under one area.

Version: 1.0
User Story:
  • As an administrator, I want to have the attributes and tags grouped under one area.
Description:
  • A property collection is a table that groups the attributes, tags, and roles in one section to be added to a class.
Prerequisite:
  • Attributes and tags created
Application:

  • A property collection can hold attributes and tags for similar type of information or content.

  • Example:

    For the property collection “General Information”, the attributes (name, PID, SKU, GTIN, Description) and tags (Brands) are brought under one property collection.

How to do: Property Collections¶

This How To guide provides instructions on how to group attributes and tags into one collection. To do this, follow these steps:



  1. Go to Settings.


  1. Click on Property Collection Configuration.


  1. Click on the “+” icon to create a property collection.


  1. Enter the name for the property collection.


  1. Increase or decrease the number of rows and columns to accommodate the attributes and tags into the property collection,


  1. Add the attributes, tags and roles to the property collection.


  1. Drag and drop the attributes and tags into the empty rows and columns.


  1. Click on Class Configuration.


  1. Click the “+” icon in the class to add the property collection.


  1. The property collection is now ready.
