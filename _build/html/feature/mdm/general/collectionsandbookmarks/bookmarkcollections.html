CORE 53: Public or Private Bookmarks and Collections¶

The Public or Private Bookmarks and Collections feature provides general users the ability to set a bookmark as a public or private bookmark.

Version: 1.0
User Story: As a user, I want to define a bookmark or collection as either Private or Public.
Description:
  • Private bookmarks are visible only to the user who created those bookmarks, whereas public bookmarks are visble to all users of the system.
  • A lock icon indicates that the bookmark is a private. bookmark
  • An unlock icon indicates that the bookmark is a public bookmark.
  • To change a bookmark into a private bookmark, user needs to click the lock icon to lock the bookmark and change it into a private one.
Prerequisite: User must create a bookmark.
Application:
  • A bookmark can be a private or a public bookmark.
  • A private bookmark will only be seen by the user who created the bookmark, and not the other users.
  • Example:

Suppose the marketing team of a company has created a bookmark for a trial, they make the bookmark as a private bookmark because they don’t want to show their bookmark to the other users of the system. On the other hand, if they want to share the bookmark with other users, they keep the bookmark as a public bnookmark.
How To A How to do: Public or Private Bookmarks and Collections document is available below.

How to do: Public or Private Bookmarks and Collections¶

This How To guide provides instructions on how to define a bookmark (or collection) to either Public or Private. A public setting grants access to all users while a private setting grants access only to specific users. Follow these steps:

Bookmarks: Setting a Public Bookmark¶


  1. Log in as admin.


  1. Go to the desired bookmark.

  1. The default setting of a bookmark is Public. Keep this setting to make a public bookmark.


  1. Once done, the bookmark can be viewed by all users.

As seen on the image above, the bookmark can be viewed by the Seller.

Bookmarks: Setting a Private Bookmark¶


  1. Log in as admin.


  1. Go to the desired bookmark.

  1. Click on the bookmark icon and set the bookmark to Private.


  1. Once done, the bookmark can be viewed only by specific users.

As seen on the image above, the bookmark cannot be viewed by the user as it is restricted to the admin.

Collections: Setting a Public Collection¶


  1. To start creating a collection, select the products or assets that will be included in the collection.

  1. Click on the pin icon on the right side of the search bar to create the collection.


  1. Name the collection and click on the Create Collection button.


  1. Once created, search for the collection as you would a bookmark.

  1. Once found, check that the default setting is Public. Keep this setting for a public collection.


  1. Once done, the collection data can be viewed by all users.

Collections: Setting a Private Collection¶


  1. To set a collection to private, search for the collection.

  1. Once searched, click on the lock icon to change the setting to Private.


  1. Once done, the collection can be viewed only by specific users.

As seen on the images above, the collection cannot be viewed by the user as it is restricted to the admin.
