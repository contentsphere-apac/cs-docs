Manage Data Transfer¶

Manage Data Transfer lets you define the inheritance of properties from PIDs to their variants.

Key Term:
  • Property: a data field which describes a product, can also use ‘attribute’.
Description:
  • There are properties that are common to a PID and its variants. Data transfer uses a coupling to push these common properties to the variants directly.
Related topics:
  • Manage PIDs
Application:
  • Manage the information that is common between the PID and its variants, and have such information applied to all variants.

How-to: Manage Data Transfer¶

Management of information between PIDs and their variants has never been this easy, and here’s how to do it.


  • Click the Administration icon.


  • Click Classes. Then, Article.

  • Select a class under Single Article.


  • Select the attributes and tags for data transfer.


  • Go to PIM.


  • Hover on the “+” icon.


  • Select Product.


  • Enter the properties for the product.


  • Click the “+” icon to add a Link Variant. Select the tag and click Next.


  • Select the variant and click Add.


  • Check if the product is added and click to transfer.


  • Notice the transfer of data to the product.
