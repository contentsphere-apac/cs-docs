Version Context¶

The Version Context feature allows a user to create a special version of a product that contains specific information intended for certain targets or channels.

Description:
  • This feature helps create various versions (varying according to the tags) of a product.
  • This feature creates clones of an original product and helps a user to run a promotional campaign on a specific version (or few versions) of a product.
Related topics:
  • Cloning
  • Manage Individual Articles
Application:

  • The Version Context can create a product version that is limited to a seasonal campaign or promotion.

  • Example:

    Retailer ‘A’ wants to run a promotion on the mobile phone’s 64 GB variant during Christmas (that will only run for that holiday season). The retailer will create a variant of the original product through the feature, Version Context and will have a time-bound campaign running for that version.

How-to: Version Context¶

To do this feature, follow these steps:


  1. Log in and go to Administration.


  1. Click on Contexts and then, click on “+” icon to create a context.


  1. Provide the following details: Name and Type. Select the Version Context option for the Type field. Click on the Create button.



  1. A new window will pop up; set the details and options: Icon, Enable Time (Campaign Period), Allow Duplicate, and the relevant Tags.


  1. Click the Save button once done. And then, click Ok.



  1. Click on Article under the Classes section. Create a new article or select an existing one from the list.


  1. Once selected, update the Version Context field that can be seen on the right. Select the newly created version context from the options and save the changes.



  1. Click on the Explore icon and go to the product.


  1. In the Overview tab, go to the section of the Version Context and click on the “+” icon to create a version of the product.



  1. Provide the necessary details and then click on the Create button.


  1. Wait for the new window, Create Clone Variant to load. Select all the Tags and Attributes that apply to that variant.


  1. Click the Create button and then, view the created variant.
