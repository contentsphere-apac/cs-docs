Manage Tag Hierarchies¶

The Manage Tag Hierarchies is an admin feature that sets up tag-based hierarchies wherein each master tag is considered as one level. This feature can mix and match tags to build simple hierarchies.

Key Term:
  • Tag: is a property that is used to describe a product (like bottle size or color).
Description:
  • Hierarchy is concerned with the classification of different objects.
  • The system allows multiple levels of hierarchies: Level 1 can be further classified by adding Level 2 and further classification of Level 2 can be done by adding extra levels under it.
Related topics:
  • Tag Management
Application:

  • Creation of hierarchy is the same as creating a Master Taxonomy. The only difference between the two is that one cannot add a Property Collection (collection of attributes and tags) in hierarchies.

  • Hierarchy basically differentiates a Parent from a Child node.

  • Example:

    If we create a hierarchy of a Master Taxonomy, Fashion (Level 1), it can be further classified to a sublevel like Shirts, Trousers, etc (Level 2). A Shirt can then be further classified based on the Material or Pattern (Level 3).

How to do: Manage Tag Hierarchies¶

To manage the tag hierarchies, follow this set of instructions:



  1. Log in and go to Administration.


  1. Go to the Data Model tab and then to Taxonomies.


  1. Under Taxonomies, click on Hierarchies.


  1. Click on the “+” icon to create a new Master Taxonomy.


  1. Fill out the required details. Also, slide the button to the right to enable the Master Taxonomy option.


  1. Click on the Create button.


Note: A Major Taxonomy has been created. Further classification can now be done by adding sublevels under it.



  1. To add sub-taxonomies, click on the newly created Master Taxonomy and wait for the column on the right to be active. Once active, click on the Add New Level button.


  1. Type the name of the sub-taxonomy and then, click on the Create ‘name’ button.


Note: The sub-taxonomy has been added and the major taxonomy had been further classified.
