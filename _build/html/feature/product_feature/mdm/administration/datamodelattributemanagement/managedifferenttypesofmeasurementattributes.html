Measurement Type: Volume¶

This feature example allows an administrator to set an attribute as Volume.

Key Term:
  • Taxonomy: A taxonomy is a scheme of classification used for an organization’s products.
Description:
  • This feature allows the setting of an attribute type as Volume.
  • This feature makes an attribute type as Volume and applies the appropriate default unit for volume (e.g. Gallon, Cubic meter, Liter).
Related topics:
  • Manage Different Types of Attributes
  • Manage Concatenated Attributes
  • Manage Calculated Attributes
Application:

  • Volume measurement type can set the volume of a liquid product (e.g. Beverages) and also set its corresponding unit.

  • Example:

    An administrator can set an attribute of a Red Bull product to volume so that a user can easily enter the volume of the product and select a unit (from a drop-down list).

How to do: Measurement Type: Volume¶

To set an attribute as a volume, follow these steps:


  1. Go to Settings.


  1. Go to the Data Model tab.


  1. Click on Measurement under the Properties.


  1. Click the ‘+’ icon to create an attribute.


  1. Enter the name of the volume attribute and click on the Create button.


  1. Check that the attribute has been added successfully.


  1. Under the measurement type, search and select Volume in the drop-down menu.



  1. Select the appropriate unit for the volume attribute in the Default Unit column (e.g. Gallon, Cubic meter, Liter).


  1. Complete the other attribute fields: Default Value, Precision, Placeholder, Description, Tooltip, Searchable, Filterable, Sortable, Availability, and Grid Editable.


  1. Click on the Save button to save the settings.
