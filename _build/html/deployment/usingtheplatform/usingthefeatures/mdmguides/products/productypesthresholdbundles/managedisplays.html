Manage Displays¶

The Manage Displays feature allows the user to sell multiple quantity products.

Key Term:
  • Loose Bundle: a group of products where the quantity of each product can be defined by the retailer.
Description:
  • The Manage Displays feature allows you to create a bundle of products grouped for the purpose of displays in store.
Related topics:
  • Manage Media Links
Application:

  • This feature brings single articles together in multiple quantities and sells them.

  • This type of bundle can be used during stock clearance, where the retailer can add a single item in multiple quantities and can sell them at an offered price.

  • Example:

    A retail company held a stock clearance sale event. With this, a retailer adds a single item in multiple quantities and sells them at a given price.

How-to: Manage Displays¶

To do this, follow these steps:


  • Go to PIM.


  • Hover over the “+” icon.


  • Click on Loose Bundle icon.


  • Enter the necessary information.




  • Click Add product(s) under Overview tab.




  • Select one or more products (button turns to blue when selected). Then, click the Add All icon.




  • Click the Back icon to view the added products.

  • To add an image, click Relationships tab.


  • Hover over on the “+” icon and click Add Entity.


  • Select the desired image and drag it to the right side pane.




  • View the created bundle in the Bundle Section of the Overview tab.
