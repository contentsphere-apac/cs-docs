================
Manage Displays
================
The Manage Displays feature allows the user to sell multiple quantity products.


====================  =================================================================================================================================================================
**Key Term:**         * **Loose Bundle:** a group of products where the quantity of each product can be defined by the retailer.

**Description:**      * The Manage Displays feature allows you to create a bundle of products grouped for the purpose of displays in store.

**Related topics:**   * :doc:`Manage Media Links </feature/product_feature/mdm/administration/datamodelrelationships/managemedialinks>`

**Application:**      * This feature brings single articles together in multiple quantities and sells them.
                      * This type of bundle can be used during stock clearance, where the retailer can add a single item in multiple quantities and can sell them at an offered price.
                      * *Example:*

                          A retail company held a stock clearance sale event. With this, a retailer adds a single item in multiple quantities and sells them at a given price.

====================  =================================================================================================================================================================


How-to: Manage Displays
=============================
To do this, follow these steps:


|

#. Go to *PIM*.

.. image:: /feature/_images/mdmimages/cs_managedisplays_gotopim.PNG
|
|


#. Hover over the "+" icon.

.. image:: /feature/_images/mdmimages/cs_managedisplays_hoverplus.PNG
|
|


#. Click on **Loose Bundle** icon.

.. image:: /feature/_images/mdmimages/cs_managedisplays_clickloose.PNG
|
|

#. Enter the necessary information.

.. image:: /feature/_images/mdmimages/cs_managedisplays_enterinformation.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_managedisplays_enterinformationsave.PNG
|
|

#. Click **Add product(s)** under *Overview* tab.

.. image:: /feature/_images/mdmimages/cs_managedisplays_clickaddproducts.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_managedisplays_clickaddproducts2.PNG
|
|

#. Select one or more products (button turns to blue when selected). Then, click the **Add All** icon.

.. image:: /feature/_images/mdmimages/cs_managedisplays_selectproductaddall.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_managedisplays_selectproductaddall2.PNG
|
|


#. Click the **Back** icon to view the added products.

.. image:: /feature/_images/mdmimages/cs_managedisplays_clickback.PNG
|

#.  To add an image, click  **Relationships** tab.

.. image:: /feature/_images/mdmimages/cs_managedisplays_addimage.PNG
|
|

#.  Hover over the "+" icon and click  **Add Entity**.

.. image:: /feature/_images/mdmimages/cs_managedisplays_hoverplusaddentity.PNG
|
|

#. Select the desired image and drag it to the right side pane.

.. image:: /feature/_images/mdmimages/cs_managedisplays_hoverplusaddentity2.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_managedisplays_hoverplusaddentity3.PNG
|
|

#. View the created bundle in the Bundle Section of the *Overview* tab.

.. image:: /feature/_images/mdmimages/cs_managedisplays_seebundle.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_managedisplays_seebundle2.PNG
|
|
