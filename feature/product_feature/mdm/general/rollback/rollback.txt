=========
Rollback
=========

The Rollback feature allows a user to switch (or to revert) to a previous version of a product in the system.

===================       ================================================================================================================================================================
**Key Term:**             * **Rollback:** a feature that enables the user to switch back to a desired version of a product.

**Description:**          * This is an enhancement to the basic *Rollback* feature, where when the user adds or removes any relationship/context to a product; it can be rolled back to the previous version.
                          * This feature allows the user *(retailer/supplier)* to compare various versions of the product, before deciding which version to rollback to.
                          * This feature allows the user *(retailer/supplier)* to preview the changes in the product versions.

**Related topics:**       * :doc:`Manage Virtual Catalogs </feature/product_feature/mdm/products/catalogmanagementvirtualproductcatalogs/managevirtualcatalogs>`
                          * :doc:`Product Classifications </feature/product_feature/mdm/governance/classificationandcategorization/improveproductclassifications>`
                          * :doc:`Compare Product Revisions </feature/product_feature/mdm/general/timelineandhistory/compareproductrevisions>`

**Application:**          * The Rollback feature is used to switch back to the desired version of a product.
                          * This is an enhancement to the basic *Rollback* feature. It allows the user add or remove any product relationship/context and return it back to the previous version.
                          * This feature allows the user *(retailer/supplier)* to compare various versions of the product, before deciding which version to rollback to.
                          * This feature allows the user *(retailer/supplier)* to preview the changes in the product versions.
                          * *Example:*

                              Retailer ‘A’ has made several changes in the product namely: deleted an image, changed the language preference, removed an attribute, etc. The Zoom in/out and compare  feature will help him examine those details across the product versions and compare these versions, before he decides to roll back to the desired one.
===================       ================================================================================================================================================================


How-to: Rollback
=====================
To do this, follow these steps:

|

#. Go to *PIM*.

.. image:: /feature/_images/mdmimages/cs_rollback_gotopim.PNG
|
|

#. Click to select the desired product for which you want to perform rollback operation.

.. image:: /feature/_images/mdmimages/cs_rollback_selectdesiredproduct.PNG
|
|

#. Click on the **Relationships** tab.

.. image:: /feature/_images/mdmimages/cs_rollback_clickrelationshiptab.PNG
|
|

#. In this demo, we will remove the image and change an attribute.

.. image:: /feature/_images/mdmimages/cs_rollback_inthisdemoremoveimage.PNG
|
|

#. Click on the **OK** button to save the changes.

.. image:: /feature/_images/mdmimages/cs_rollback_clickOKimage.PNG
|
|

#. In the *Property Collection* Tab, change the name (or any other attribute) of the product.

.. image:: /feature/_images/mdmimages/cs_rollback_inthepropertycollection.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_rollback_changeattribute.PNG
|
|


#. Click **Save** to save the changes on the product.

|

#. Click the **Timeline** tab. The current version of the product would be with the latest version number on the top.

.. image:: /feature/_images/mdmimages/cs_rollback_clicktimelinetab.PNG

|
|

#. Click on the **Zoom in** icon to preview the changes in the product versions.

.. image:: /feature/_images/mdmimages/cs_rollback_clickzoomin.PNG
|
|

#. Select the versions (one or more) that you would like to compare the current version to.

.. image:: /feature/_images/mdmimages/cs_rollback_selectversion.PNG
|
|

#.  Click on the **Compare with current version** icon.Then, click **OK** to close the window.

.. image:: /feature/_images/mdmimages/cs_rollback_clickcomparecurrent.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_rollback_clickcomparecurrent2.PNG
|
|

#. Click the **Rollback** icon of the selected version.

.. image:: /feature/_images/mdmimages/cs_rollback_clickrollback.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_rollback_clickrollback2.PNG
|
|

#. Once the rollback has been confirmed, go to  the *Properties* Tab, and the changes are restored. View the image (now reverted to the old version).

.. image:: /feature/_images/mdmimages/cs_rollback_rollbackconfirmed.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_rollback_rollbackconfirmed2.PNG
|
|
