=====================
Create a New Product
=====================
The system allows effective management of products in the system. Products can be created, deleted, and edited.


How-to: Create New Products
==============================
To do this, follow these steps:

|

#.	Click  **Explore** from the top right and go to *PIM*.

.. image:: /feature/_images/mdmimages/cs_createnewproduct_clickexplore.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_createnewproduct_clickexplore2.PNG
|
|


#. Hover over the "+" icon and select the desired product (in this demo, we select **Smartphone Accessories**).

.. image:: /feature/_images/mdmimages/cs_createnewproduct_hoverplus.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_createnewproduct_hoverplus2.PNG
|
|

#. Enter the necessary information.

.. image:: /feature/_images/mdmimages/cs_createnewproduct_enternecessaryinfo.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_createnewproduct_enternecessaryinfo2.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_createnewproduct_enternecessaryinfo3.PNG
|
|

#. Go back to *Overview* tab and click **Add Taxonomy** to add taxonomy to the product.

.. image:: /feature/_images/mdmimages/cs_createnewproduct_gobackoverview.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_createnewproduct_gobackoverview2.PNG
|
|

#. Taxonomy menu appears. Select the desired taxonomy (in this demo, we select **Accesories**).

.. image:: /feature/_images/mdmimages/cs_createnewproduct_taxmenuappears.PNG
|
|

#. View the selected taxonomy.

.. image:: /feature/_images/mdmimages/cs_createnewproduct_viewselectedtax.PNG
|
|

#. Click  **Relationships** tab to add an image to the product.


.. image:: /feature/_images/mdmimages/cs_createnewproduct_clickrelationshiptab.PNG
|
|

#. Click on  "+" to add an entity.


.. image:: /feature/_images/mdmimages/cs_createnewproduct_clickplus.PNG
|
|

#. Click the **select** checkbox to add the desired image (button turns to blue when selected), then click  **Add**.

.. image:: /feature/_images/mdmimages/cs_createnewproduct_clickselect.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_createnewproduct_clickselect2.PNG
|
|

#. Drag the selected image to the right pane can also be performed to add the image.

.. image:: /feature/_images/mdmimages/cs_createnewproduct_dragimage.PNG
|
|

#. To add more image, click **Back**, and repeat the previous steps.

.. image:: /feature/_images/mdmimages/cs_createnewproduct_addimageclickback.PNG
|
|

.. image:: /feature/_images/mdmimages/cs_createnewproduct_addimageclickback2.PNG
|
|

|
